2. Внедрение решения mySAP Business Suite

Введение

    Фаза 1. Подготовка проекта

    Фаза 2. Концептуальный проект

    Фаза 3. Реализация

    Фаза 4. Заключительная подготовка

Введение

        SAP Solution Manager разработан и структурирован для поддержки на всех этапах проекта внедрения. Эти этапы основываются на 5 фазах маршрутной карты внедрения:

  1. Подготовка проекта;
  2. Концептуальный бизнес-проект;
  3. Реализация;
  4. Окончательная подготовка;
  5. Переход к продуктивной эксплуатации и поддержка.

        В нашем курсе мы рассмотрим первые три фазы и создание электронных учебных материалов на четвертой фазе (см. рис. 2.1).

   

Рисунок 2.1 – Внедрение решения mySAP Business Suite.

        Как видно из рисунка SAP Solution Manager используется для выполнения следующих работ по проекту:

          Подготовка проекта: определение ландшафта проекта и системы.

        Концептуальный бизнес-проект: определение клиентского решения, основанного на предварительно определенных сценариях и сконфигурированных бизнес-процессах; документирование бизнес-сценариев и бизнес-процессов предприятия.

        Реализация: конфигурация, сравнение и распределение параметров настройки; организация и проведение тестов.

        Окончательная подготовка: обучение пользователей, перенос данных.

        Помимо задач по фазам проекта внедрения в SAP Solution Manager поддерживаются общие функции, такие как администрирование проекта, предоставление отчетности в любой момент для получения информации о статусе проекта и динамике тестирования и конфигурации. Маршрутная карта сопровождает команду проекта внедрения по этапам проекта.
